## Comparação de Fmax para o totalizador_b

Como o atraso foi de cerca de 6ns

Tmin = 6 ns

Fmax_gate = 1 / Tmin = 1 / 6e-9 = 166,67 MHz

Considerando Fmax do Quartus = 468,6 MHz:

DeltaF = 468,6 - 166,67 = 301,93 MHz

Diferença percentual = |468,6 - 166,67| / 468,6 x 100 = 64,43%

Conclusão: na simulação gate-level, a frequência máxima estimada ficou em 166,67 MHz, cerca de 64,43% abaixo da Fmax reportada pelo Quartus.
